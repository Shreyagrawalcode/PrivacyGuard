# PrivacyGuard
An Android cybersecurity application focused on helping users understand and manage app permissions on their devices.

## Status
In development — permission scanner feature in progress.

## Planned Features
- Scan installed apps and display their permissions
- Flag apps with excessive or suspicious permissions
- Provide plain-language explanations of what each permission means
- Alert users to potential privacy risks

## Tech Stack
- **Language:** Kotlin
- **UI Framework:** Jetpack Compose (Material 3)
- **Platform:** Android (min SDK 24 / Android 7.0+)
- **IDE:** Android Studio

## Current State
Architecture and UI scaffolding complete. MainActivity set up with Jetpack Compose. 
Permission scanner screen (AppScannerScreen.kt) created — feature implementation in progress.

## Why I Built This
Cybersecurity students learn to identify vulnerabilities in systems. 
This project applies that thinking to everyday Android devices — 
most users have no visibility into what permissions their apps are silently using.
